Välkommen till Responsiv Design

Vad är responsiv design?

Responsiv design innebär att skapa webbsidor som fungerar och ser bra ut på alla enheter, oavsett skärmstorlek.

CSS

Flexbox

Flexbox är en layoutmodell som gör det enkelt att skapa flexibla och responsiva layout-strukturer.

Grid

Grid är ett kraftfullt verktyg för att bygga komplexa och responsiva layouter.

Media Queries

Med media queries kan du anpassa designen för olika skärmstorlekar och enheter.
